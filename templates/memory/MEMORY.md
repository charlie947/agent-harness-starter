## How to use this index

Every memory file gets one line here. Format:

```
- [short-name](filename.md) — one-line hook
```

Keep the index under 200 lines total. If it grows past that, move details into topic files and let this stay a pure pointer list.

## Memory types

- **user_** — facts about you (role, preferences, knowledge)
- **feedback_** — corrections / preferences the user has given
- **project_** — facts about ongoing work
- **reference_** — pointers to external systems
